# Практическая работа №2: Основы работы с PHP
---

## 📋 Задания

### **Задание 1** – Вывод HTML через PHP
- Вся разметка выведена через инструкцию `echo`
- Использовано чередование кавычек (`'` и `"`) для избежания конфликтов

### **Задание 2** – Вывод диалога (7 фрагментов)
- Выведен диалог Понтия Пилата и Иешуа из «Мастера и Маргариты»
- Использовано 7 инструкций `echo` (по одной на фрагмент)
- Выполнено HTML-форматирование: абзацы (`<p>`), жирный шрифт (`<b>`), переносы (`<br>`)

### **Задание 3** – Переменные для хранения диалога
- Созданы строковые переменные для каждого фрагмента
- Использовано две пары PHP-тегов: инициализация переменных и вывод

### **Задание 4** – Арифметические вычисления
- Вычислено выражение: `(952 / 4) * 3 - (476 / 7) + 196`
- Все операнды сохранены в переменные
- Результат: **842**

### **Задание 5** – Конвертация валюты
- Переведено 1000 долларов в юани через рубли
- Курсы: 1 USD = 75.87 RUB, 1 RUB = 0.09 CNY
- Результат: **1000 USD = 6828.3 CNY**

### **Задание 6** – Переменные → Константы
- Все переменные заменены на константы (`define()`)
- Константы названы в верхнем регистре

### **Задание 7** – Константа-массив
- Создана константа-массив с данными о книге
- Вывод выполнен с использованием тега `<pre>`

### **Задание 8** – Константа-массив для альбома
- Данные об альбоме Pink Floyd объединены в константу-массив
- Вместо 6 отдельных переменных — одна структурированная константа

### **Задание 9** – Полный вывод через PHP
- Весь HTML-код страницы выведен через `echo`
- Нет закрывающих PHP-тегов до конца файла

### **Задание 10** – Слабая типизация PHP
- Протестированы арифметические действия с разными типами данных
- Проанализировано автоматическое преобразование типов

### **Задание 11** – var_dump() и типы данных
- Проверка утверждения: «Тип данных зависит от способа представления»
- Использована конструкция `var_dump()` для определения типов

### **Задание 12** – Вывод с одинарными кавычками
- Диалог выведен с использованием одинарных кавычек
- Внутренние кавычки экранированы (`\'`)

### **Задание 13** – Вывод с двойными кавычками
- Диалог выведен с использованием двойных кавычек
- Внутренние кавычки не требуют экранирования

### **Задание 14** – HEREDOC синтаксис
- Вывод диалога с использованием HEREDOC
- Применена интерполяция переменных

### **Задание 15** – Сборка сайта
- Создан многостраничный сайт «Музыкальный сервис»
- Страницы: Главная, О нас, Контакты, Группы, Альбомы, Треки, Консоль
